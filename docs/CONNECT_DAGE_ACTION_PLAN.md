## Day 1 · 立即可做（30 分钟 · 现金 ¥260）

### 步骤 1.1 · 阿里云注册（5 分钟）
1. 浏览器打开 https://www.aliyun.com
2. 右上角"免费注册" · 用你常用手机号
3. 实名认证（个人即可 · 上传身份证）

### 步骤 1.2 · 开 ECS 服务器（10 分钟 · ¥260）
1. 登录后顶部搜索"ECS"
2. 点"立即购买"
3. 选项：
   - **付费方式**：包年包月（先 1 个月试 · 之后包年 7.5 折）
   - **地域**：**华东 1（杭州）** 或 **华东 2（上海）** · 必须！客户在大陆
   - **实例**：ecs.t6-c1m2.large（**4C 8G**）· 月费约 ¥230
   - **镜像**：CentOS 8.4（64 位）
   - **系统盘**：50G SSD（含）· 加 100G 数据盘 · ¥30
   - **公网带宽**：按使用流量 → 改成"按固定带宽 5M" · 月 ¥45
   - **安全组**：开放 80 / 443 / 22（这 3 个端口）
4. 设置 root 密码 · **存到密码管理器**
5. 提交订单 · 付 ¥260
6. 5 分钟后 · 服务器就绪 · 记下 **公网 IP**（发给童虎）

### 步骤 1.3 · 域名注册（10 分钟 · ¥55/年）
1. 阿里云顶部搜"域名"
2. 输入想要的：例 `wechatagent.com` 或 `xinxinai.com`（5-10 字母）
3. 选 .com（最贵但最专业 · ¥55/年）
4. 加入清单 → 立即购买
5. 填**域名持有者信息**（个人即可 · 备案要用 · 名字/身份证/手机/邮箱）

### 步骤 1.4 · 通知童虎
微信发 / 飞书发：
- 公网 IP：xxx.xxx.xxx.xxx
- root 密码：xxx
- 域名：xxx.com
- 完成时间：xxx

童虎收到后立即写 deploy/init.sh 适配你的服务器。

## Day 2-3 · 资源准备（1 小时 + 等审核 · ¥150 充值）

### 步骤 2.1 · 提交 ICP 备案（10 分钟 · 等 7-15 天）
1. 阿里云控制台 · 左侧"备案" → "新增备案"
2. 选"个人备案"（如果你还没工商户）· 后续工商户可改企业
3. 上传身份证 + 域名证书 + 服务器信息
4. 提交后 → 拍照留底 → 等阿里云初审 1-3 天 · 通信管理局 7-15 天
5. **备案期间域名暂不能用** · 但可以并行做其他事

### 步骤 2.2 · 申请 4 个 LLM API key（30 分钟 · 充值 ¥150）

#### A · MiniMax 极速版（已有 · 跳过）

#### B · 豆包 1.5 Pro（必申 · 中文拟人冠军 · 60% 场景）
1. 浏览器打开 https://console.volcengine.com/ark
2. 注册 · 实名认证
3. 创建"推理接入点" → 选 doubao-1-5-pro-32k 模型 → 拿 endpoint_id
4. 控制台 → API Key 管理 → 创建 → 拿 API key
5. **复制保存**：
   ```
   DOUBAO_API_KEY=xxx
   DOUBAO_ENDPOINT_ID=ep-xxx
   ```
6. 充值 ¥50（送一些额度）

#### C · DeepSeek V3.2（必申 · 性价比王 · 25% 场景）
1. 浏览器打开 https://platform.deepseek.com/api_keys
2. 注册手机号（注册送 ¥10）
3. 创建 API Key → 拿 sk-xxx
4. 充值 ¥50（约够 5000 客户用 1 个月）

#### 完成后 · 全部发给童虎
童虎填进 `.env` · 重启 server · 真接通 4 模型路由。

### 步骤 2.3 · 律师询价（30 分钟 · 决定一家 ¥3-8K）
1. 淘宝搜"SaaS 用户协议起草"
2. 看评价 ≥4.8 · 销量 ≥100 单

5. 选最专业的一家（不一定最便宜）
6. **预算 ¥3000-8000** · 周期 1-2 周

## Day 4-7 · 备案等的同时做（并行）

### 步骤 3.1 · 个体工商户（关键 · 公司主体兜底）
1. 微信搜"个体工商户注册"小程序（阿里云/微信/支付宝都有）
2. 准备：身份证 · 经营地址（家庭地址即可）· 经营范围（"信息技术服务"）
3. 提交申请 · 7-15 天下证（部分地区当天）
4. 拿到营业执照后：
   - 拍照保存
   - 后续微信支付商户号 + 法务签字 + 银行对公账户都要用
5. **预算 ¥0-500**（部分地区收工本费）

### 步骤 3.2 · 微信支付商户号（拿到工商户后做 · 等 1-2 周）
1. https://pay.weixin.qq.com/
2. "立即接入" → "个体工商户"
3. 上传：营业执照 + 身份证 + 银行对公账户
4. 7-15 天审核 · 下来后拿到：
   - mch_id（商户号）
   - api_v3_key（密钥）
   - 证书文件（pfx）
5. **全部发给童虎** · 童虎接入 server/billing.py 真扣费

### 步骤 3.3 · 找 5 个种子客户（最重要！）
1. 列你认识的人里**用微信工作的**（微商/销售/客服/接单/中介）

3. 收集 ≥5 个意向客户名单（行业 / 微信号 / 现在用什么工具）
4. 这 5 个是**第一波内测** · 帮你打磨产品 + 写第一批好评

### 步骤 3.4 · Windows 机器准备（你自己用 · 当 0 号客户）
1. 任何 Windows 10/11 PC（你家那台台式 / 旧笔记本都行）

## Day 8-14 · 备案下来 · 部署上线（童虎主导）

### 步骤 4.1 · 域名 DNS 配置（5 分钟 · 你做）
1. 备案下来通知后 · 阿里云控制台 → 域名 → DNS 解析
2. 加 A 记录：
   - 主机：`api`（或 `@`）
   - 记录值：你 ECS 的公网 IP
3. 等 5 分钟生效

### 步骤 4.2 · 童虎远程登录服务器部署（30 分钟 · 童虎做）
1. 你提供 SSH 信息（IP + root + 密码）
2. 童虎执行：
   ```bash
   # 1. 装 docker
   curl -fsSL https://get.docker.com | sh
   
   # 2. clone 代码
   git clone <wechat_agent>
   cd wechat_agent
   
   # 3. 填环境变量（API key + 域名）
   cp .env.example .env
   nano .env
   
   # 4. 一键启动（已写好）
   bash deploy/init.sh
   ```
3. 完成后 · https://你的域名/v1/health 返回 200 = 成功

### 步骤 4.3 · 第一个种子客户安装（30 分钟 · 童虎远程指导）
1. 你后台发激活码：
   ```
   POST /admin/issue_code → WXA-2026-A1B2-C3D4-E5F6
   ```
2. 把这个码 + setup.exe 下载链接发给种子客户
3. 远程协助客户安装（TeamViewer 或微信视频）：
   - 双击 setup.exe → 同意协议 → 输激活码 → 完成
   - 桌面图标自动启动 → 系统托盘绿灯
4. 客户用一会儿 → 看 dashboard 数据 → 反馈

### 步骤 4.4 · 监控 + 调优（持续）
1. 浏览器打开 https://你的域名/admin（输你的 admin token）
2. 看：客户列表 · 健康分 · 今日成交 · 错误日志
3. 任何异常 → 截图发童虎

## 🚨 遇到问题 SOP

### 技术问题（找童虎）
- server 报错 · 客户装不上 · API 调不通 · 数据库慢 → **找童虎**
- 截图 / 错误日志 / 操作步骤 一起发

### 法律问题（找律师）
- 客户提退款纠纷 · 收到投诉 · 收到律师函 → **第一时间找律师**
- 不要自己回应客户法律问题

### 商务问题（你决策）
- 定价调整 · 渠道合作 · 客户拒绝 → **你自己拍**
- 咨询童虎给数据 · 但决策你做

### 不要做的事
- ❌ 不要自己改代码（让童虎改）
- ❌ 不要在客户群说"全自动"（用"AI 副驾驶"）
- ❌ 不要接灰产客户（赌/色/诈/医诊/金融荐股）· 一个就毁所有
- ❌ 不要免费送给陌生人（每个客户单独发激活码）
- ❌ 不要把激活码发朋友圈（一码一机绑定 · 但容易被刷）
