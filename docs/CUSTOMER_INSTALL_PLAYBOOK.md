## 🔧 Mac 端准备（童虎执行 · 你看）

### 配置 MiniMax
```bash
cd /Users/lian/wechat_agent
echo "MINIMAX_API_KEY=sk-cp-你的key" >> .env
echo "BAIYANG_HERMES_MOCK=false" >> .env
```

### 启动 server
```bash
source .venv/bin/activate
uvicorn server.main:app --host 0.0.0.0 --port 8327 --reload
```

### 启动 ngrok（临时公网 URL）
```bash
# 首次装 ngrok
brew install ngrok
ngrok config add-authtoken <你的 token · ngrok.com 免费注册拿>

# 暴露 server
ngrok http 8327
```
ngrok 启动后给你一个 `https://xxxx.ngrok.io` · 这就是**客户端 server URL**。

### 生成激活码（后台）
```bash
curl -X POST http://localhost:8327/admin/issue_code \
  -H "X-Admin-Token: $BAIYANG_ADMIN_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"plan": "pro", "valid_days": 365}'
# 返回：{"code": "WXA-2026-A1B2-C3D4-E5F6", ...}
```

## 🪟 Windows 端 · install_client.ps1 模板

> 童虎会根据你提供的 ngrok URL + 激活码 · 填好给你
> 你复制粘贴到客户 PowerShell 就行

```powershell
# wechat_agent 客户端 · 一键安装（管理员 PowerShell 运行）
# Generated by 童虎 · 2026-04-17

$ErrorActionPreference = "Stop"

Write-Host "[1/7] 装 Python 3.11..." -ForegroundColor Cyan
# 用 winget（Windows 11 自带）· Windows 10 用 choco
if (Get-Command winget -ErrorAction SilentlyContinue) {
    winget install --id Python.Python.3.11 -e --accept-source-agreements --accept-package-agreements
} else {
    iwr https://community.chocolatey.org/install.ps1 -UseBasicParsing | iex
    choco install python311 -y
}

Write-Host "[2/7] 刷新环境变量..." -ForegroundColor Cyan
$env:Path = [System.Environment]::GetEnvironmentVariable("Path","Machine") + ";" + [System.Environment]::GetEnvironmentVariable("Path","User")

Write-Host "[3/7] 装依赖..." -ForegroundColor Cyan
python -m pip install --upgrade pip
python -m pip install fastapi uvicorn pydantic sqlalchemy aiosqlite aiohttp pyyaml wxautox humancursor

Write-Host "[4/7] 拉取 wechat_agent client..." -ForegroundColor Cyan
$installDir = "$env:USERPROFILE\wechat_agent"
if (Test-Path $installDir) {
    Remove-Item -Recurse -Force $installDir
}
# 源码下载（未来 setup.exe · 现在直接 git clone）
git clone https://github.com/YOUR_REPO/wechat_agent.git $installDir
cd $installDir

Write-Host "[5/7] 写 .env 配置..." -ForegroundColor Cyan
@"
BAIYANG_SERVER_URL=https://xxxx.ngrok.io
BAIYANG_ACTIVATION_CODE=WXA-2026-XXXX-XXXX-XXXX
BAIYANG_TENANT_ID=tenant_客户名
BAIYANG_HERMES_MOCK=false
"@ | Out-File -Encoding utf8 "$installDir\.env.client"

Write-Host "[6/7] 创建桌面快捷方式 + 开机自启..." -ForegroundColor Cyan
$WshShell = New-Object -ComObject WScript.Shell
$shortcut = $WshShell.CreateShortcut("$env:USERPROFILE\Desktop\wechat_agent.lnk")
$shortcut.TargetPath = "$installDir\client\start.bat"
$shortcut.Save()

# 开机自启（注册表）
$regPath = "HKCU:\SOFTWARE\Microsoft\Windows\CurrentVersion\Run"
Set-ItemProperty -Path $regPath -Name "WechatAgent" -Value "$installDir\client\start.bat"

# 创建启动脚本
@"
@echo off
cd /d $installDir
python -m client.main --env .env.client
"@ | Out-File -Encoding ascii "$installDir\client\start.bat"

Write-Host "[7/7] 启动客户端..." -ForegroundColor Cyan
Start-Process "$installDir\client\start.bat"

Write-Host ""
Write-Host "✅ 安装完成！" -ForegroundColor Green
Write-Host "桌面有 wechat_agent 图标 · 下次开机自动启动"
Write-Host "系统托盘绿灯 = 正常运行"
Write-Host ""
Write-Host "下一步：让客户用手机微信发一条消息给自己 PC 微信 · 看 AI 自动回"
```

## 🚨 常见问题快速处理

### Q1 · "Python 装不上"
```
原因：Windows 版本老 / 权限不够
解决：
  - Win10 需升级到 21H2+
  - 用 choco 安装（童虎给命令）
  - 管理员权限运行 PowerShell
```

### Q2 · "pip install 超时"
```
原因：国内网络到 PyPI 慢
解决：改清华镜像
  pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple
```

### Q3 · "wxauto 不工作"
```
原因：微信版本不兼容
解决：
  - 客户装微信 4.0+
  - 童虎切版本（wxauto-4.0 或 wxautox Plus）
```

### Q4 · "客户端连不上 server"
```
原因：ngrok URL 过期 / 防火墙
解决：
  - 重新启 ngrok · 拿新 URL
  - 改 .env.client 里的 URL
  - 重启 client.main
```

### Q5 · "客户担心数据安全"
```
话术：
  "你原始聊天随时可以导出（我们有 data_export API）
  30 天不满意全删 · 数据加密
  我们签了 v3 协议 · 法律保障"
```
