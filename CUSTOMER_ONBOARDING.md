## 客户首次操作

```
1. 装 wechat_agent.exe (5MB, 桌面快捷)
2. 启动 → 桌面弹二维码窗口
3. 手机微信"扫一扫"扫码
4. 微信弹"登录确认" → 点确认
5. 完成 (后续自动)
```

**真特别提醒**:
- 扫码用客户**主用手机**的微信号
- 扫码后 **客户机不能关机** (关机 = 服务暂停)
- 第 1 周客户**不能用同一手机加群**
- 出问题立即截图 + 桌面 wxagent_*.txt 发回技术

## 真异常处理

| 现象 | 真处理 |
|------|--------|
| 微信弹"未授权设备" | iPad 协议正常提示, 点确认即可 |
| 客户被封号 | 客户自担风险 (用户协议明确), 1-3 天后再扫 |
| WeChatPadPro 挂 | 服务自动切 Plan B (wxauto/OCR, 待实施) |
| 客户机断网 | 服务暂停, 网恢复自动重连 |
